#  Numpy
## Numpy 란?
1. Python의 고성능 과학 계산용 라이브러리
2. 선형대수의 표현법을 코드로 처리
## 넘파이 특징
1. 속도가 빠르고 메모리 사용이 효율적
* 데이터를 메모리에 할당하는 방식이 기존과 다르기 때문

2. 반복문을 사용하지 않음
* 연산 시 병렬처리
* 함수를 한번에 많은 요소에 적용

3. 다양한 선형대수 관련 함수 제공
* C, C++등 다른언어와 통합 사용 가능
